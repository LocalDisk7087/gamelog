# Gamelog App (Front-End)

This is the front-end (React.js) portion of the Gamelog App project. This project tracks a video game backlog using the MERN stack.

## Technology Stack

* **React.js**
* **React-Bootstrap** (For UI components)
* **Axios** (For API requests)

## Related Repositories

* **Back-End (Node.js/Express):** <https://github.com/LocalDisk7087/gamelog-backend>

## How to Run Locally

1.  Clone the repository.
2.  Run `npm install`
3.  Run `npm run dev`
